## 1. Password Reset Policy

All password reset requests must be submitted through the IT Support portal or by calling the IT helpdesk.

### Process

- Verify your identity using your employee ID and security questions.
- Password resets are typically completed within 15 minutes during business hours.
- You will receive a temporary password via SMS or email.
- You must change the temporary password on first login.
- New passwords must meet complexity requirements:
  - Minimum 12 characters
  - Include uppercase, lowercase, numbers, and special characters
  - Cannot reuse last 5 passwords

### Self-Service Option

- Employees can reset passwords through the self-service portal at [https://password.contoso.com](https://password.contoso.com).
- Requires prior enrollment in multi-factor authentication.

### Emergency Resets

- After-hours emergencies: Call +1-800-555-HELP.
- Critical system access: Contact your manager for escalation.

## 3. Hardware Troubleshooting

### Printers

#### Common Issues

- Paper jams: Clear paper path and check for torn pieces.
- Not printing: Check connection, restart printer, and verify queue.
- Poor print quality: Run cleaning cycle and check ink/toner levels.
- Color issues: Calibrate printer and check color settings.

**Escalation:** Contact IT helpdesk if issue persists after basic troubleshooting.

### Monitors

#### Common Issues

- No display: Check cable connections and power source.
- Flickering: Check refresh rate settings and try a different cable.
- Wrong resolution: Adjust display settings in the operating system.
- Dead pixels: Document and report for warranty replacement.

### Keyboards/Mice

#### Common Issues

- Not responding: Check USB connection and try a different port.
- Wireless not working: Replace batteries and re-pair device.
- Keys not working: Clean keyboard and check for physical damage.
- Erratic mouse movement: Clean sensor and try a different surface.

### Laptops

#### Common Issues

- Won't power on: Check battery charge and try AC adapter only.
- Overheating: Ensure vents are clear and use on a hard surface.
- Slow performance: Close unnecessary programs and restart device.
- Battery not charging: Check AC adapter and update BIOS.

### Hardware Replacement

- Submit request through IT Support portal.
- Include device serial number and problem description.
- Standard replacements: 2-3 business days.
- Loaner equipment available for urgent needs.

## 4. Network and VPN Access

### VPN Access

- All remote workers must use company VPN.
- Download VPN client from IT portal.
- Use your network credentials to connect.
- Report connection issues immediately.

### Network Troubleshooting

- Check physical connections.
- Restart network adapter.
- Run network diagnostics tool.
- Contact IT helpdesk if issue persists.

### WiFi Issues

- Preferred network: Contoso-Corporate
- Guest network: Contoso-Guest, with limited access
- Conference rooms: Contoso-Meeting
- Report dead zones to IT for coverage improvement.

## 5. Email and Communication Tools

### Email Issues

- Cannot send/receive: Check internet connection and verify account status.
- Mailbox full: Archive or delete old emails.
- Spam/phishing: Do not click links; forward to security@contoso.com.
- Out of office: Configure through email client settings.

### Microsoft Teams

- Preferred communication tool for internal messaging.
- Video calls: Ensure camera/microphone permissions are enabled.
- Screen sharing: Use **Share specific window** for sensitive data.
- Recording meetings: Only with participant consent.

## 6. Security and Access Management

### Multi-Factor Authentication (MFA)

- Required for all accounts.
- Enroll through [https://mfa.contoso.com](https://mfa.contoso.com).
- Supported methods: Authenticator app, SMS, phone call.
- Backup codes available for emergencies.

### Access Requests

- Submit through IT Security portal.
- Manager approval required.
- Follows principle of least privilege.
- Access review every 90 days.

### Locked Accounts

- Automatically locked after 5 failed login attempts.
- Self-service unlock: [https://unlock.contoso.com](https://unlock.contoso.com).
- IT helpdesk can unlock immediately if needed.

## 7. Data Backup and Recovery

### Automatic Backups

- Desktop/laptop files: Backed up nightly to cloud storage.
- Network drives: Continuous backup.
- Email: Retained for 7 years.

### File Recovery

- Recent files: Check recycle bin first.
- Deleted files: Submit recovery request, available for 30 days.
- Older files: Contact IT for archive retrieval, 2-3 business days.

## 10. Ticket Submission Guidelines

When submitting a ticket, please include:

- Your name and employee ID
- Location/office
- Contact phone and email
- Detailed description of the issue
- Any error messages, screenshots helpful
- Steps to reproduce the problem
- When the issue started
- What you've already tried

### Priority Levels

- Critical: System completely down or security incident
- High: Major functionality unavailable or multiple users affected
- Medium: Partial functionality issue with workaround available
- Low: Minor inconvenience or information request
