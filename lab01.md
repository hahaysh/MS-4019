# 데모 1. Copilot Chat 및 Prebuilt Agent 활용

## 목표

- Copilot Chat 기본 화면과 사용법 이해
- 대화형 AI와 일반 검색의 차이 확인
- 웹 정보 + 문서를 결합한 리서치 및 이벤트 기획 경험
- 파일 업로드 기반 분석·요약·생성 경험
- Copilot Pages를 이용한 협업 경험
- 에이전트 스토어와 미리 빌드된 Agent(PowerPoint, Prompt Coach) 활용 경험

---

## 실습 대상 환경

- Microsoft 365 Copilot 라이선스 없이 Copilot Chat 사용 가능 (회사 계정 기준)
- 엔터프라이즈 데이터 보호가 적용되는 회사 계정으로 진행
- 데모는 **일반 채팅**(임시 채팅 아님)으로 진행
- 이미지 생성 / PowerPoint 에이전트는 리소스·라이선스에 따라 사용 횟수가 제한될 수 있음 → 시간이 오래 걸리는 작업은 미리 만들어 둔 결과로 진행 권장
- 접속 URL

```text
https://m365.cloud.microsoft/
```

## 데모 1-0. Copilot Chat 화면 둘러보기

### 시나리오

첫 질문 전에 Copilot Chat의 기본 화면과 주요 메뉴를 빠르게 확인

### 강사 진행

- **왼쪽 메뉴**: 새 채팅 / 검색 / 라이브러리, 그 아래에 내가 사용·생성한 에이전트 표시
- **Microsoft 365 앱 이동**: Outlook, Teams, Word, Excel, PowerPoint 등으로 바로 이동
- **응답 방식 선택**: 기본값 `자동`, 빠른 답변은 `신속한 응대`, 깊은 답변은 `깊이 생각하기` / 모델 선택 가능(실습은 기본 설정)
- **임시 채팅 vs 일반 채팅**: 임시 채팅은 기록을 남기지 않는 일회성, 맥락 유지·재참고가 필요하면 일반 채팅
- **엔터프라이즈 데이터 보호**: 회사 계정 데이터는 기업 보안 정책의 보호를 받음
- **점점점(⋯) 메뉴**: 최근 Copilot Pages 다시 열기, 피드백 보내기, 설정, 빠른 도움말
- **설정 > 데이터 컨트롤**: 조직 내 프롬프트 표준화·공유에 활용

### 설명 포인트

- 화면은 단순하지만 대화형 AI의 진입점
- Copilot은 Microsoft 365 서비스와 함께 쓸 때 강점이 큼
- 회사 계정 사용 시 기업용 보안·규정이 그대로 적용

## 데모 1-1. Copilot Chat 기본 사용

### 시나리오

Copilot Chat이 일반 검색과 어떻게 다른지 확인 (대화 컨텍스트 유지 + 후속 질문)

### 강사 진행

- 새 채팅으로 첫 질문을 하고, 후속 질문으로 대화를 이어감
- 두 개의 예제 세션을 각각 새 채팅으로 진행

#### 예제 A — 새 채팅

```text
Microsoft 365 Copilot의 주요 기능을 5가지로 요약해줘.
```

후속 Prompt

```text
표 형식으로 정리해줘.
```

후속 Prompt

```text
IT 관리자가 알아야 할 내용만 다시 정리해줘.
```

#### 예제 B — 새 채팅

```text
Microsoft 365 Copilot의 보안 관련 기능을 3가지로 요약해줘.
```

후속 Prompt

```text
IT 헬프데스크 신입용 체크리스트로 바꿔줘.
```

### 설명 포인트

- 검색이 아닌 대화형 AI — 맥락을 유지하며 답변 생성
- 후속 질문으로 결과를 원하는 형태(표, 체크리스트)로 변형
- 검색 메뉴는 답변 생성이 아니라 과거 대화를 다시 찾는 용도

## 데모 1-2. 웹 정보로 산업 동향 요약 (서밋 기획 시작)

### 시나리오

최신 웹 정보를 활용해 산업 트렌드를 요약하고 고객 혁신 서밋 아이디어로 발전

### 강사 진행 — 새 채팅

Prompt (웹 트렌드 조사)

```text
전자산업에서 2025년까지 가장 주목받을 혁신 트렌드 세 가지는 무엇이며, 이러한 트렌드를 바탕으로 고객 혁신 서밋의 의제를 어떻게 구성할 수 있을까요?
```

후속 Prompt (세션 아이디어)

```text
위의 트렌드를 기반으로 고객 혁신 서밋을 위한 매력적인 세션 제목 5개를 제안하고, 각 세션에 대한 간략한 설명을 작성해 주세요.
```

후속 Prompt (톤 개선)

```text
설명은 활기차고 전문적인 어조를 사용하여 더욱 설득력 있게 작성해 주세요.
```

### 설명 포인트

- 단순 검색이 아니라, 검색한 내용을 바로 활용 가능한 결과로 생성
- 처음부터 다시 쓰지 않고 후속 프롬프트로 계속 개선

---

## 데모 1-3. 브레인스토밍 및 세션 아이디어 초안

### 시나리오

정리된 트렌드를 실제 행사 기획에 쓸 수 있는 세션 초안으로 발전 (그라운딩 데이터 활용)

### 강사 진행

Prompt (트렌드 기반 세션 제안)

```text
위의 트렌드를 기반으로 고객 혁신 서밋을 위한 매력적인 세션 제목 5개를 제안하고, 각 세션에 대한 간략한 설명을 작성해 주세요.
```

Prompt (이미 정해진 세션을 그라운딩 데이터로 제공)

```text
아래에 발표자가 이미 정해진 세션이 있습니다.
고려해서 세션 제목 6개를 제안해주고, 각 세션에 대한 설명 작성해주세요.
---------------
아래는 정해진 세션 :
1. 키노트(본부장님) : 반도체 산업에서의 온톨로지 혁명
2. 사이드 세션(전략실) : 불확실성의 시대, 시장 리더는 어떻게 만들어지는가
```

### 설명 포인트

- 이미 정해진 주제가 있으면 그라운딩 데이터로 제공해 반영
- 아이디어 나열을 넘어 발표·행사 기획에 쓸 초안까지 빠르게 생성

---

## 데모 1-4. 의제 타임라인 시각화 및 로고 생성

### 시나리오

세션 아이디어를 시간대별 일정표로 정리하고, 서밋 로고 초안까지 생성

### 강사 진행

Prompt (일정표 표 만들기)

```text
반도체 트렌드를 주제로 한 1일 서밋의 일정표를 작성해 주세요.
서론, 마무리, 오전 9시, 11시, 오후 1시 30분, 3시 세션,
두 번의 짧은 휴식 시간, 그리고 1시간의 점심 시간을 포함하고,
테이블 형식으로 작성해 주세요.
```

Prompt (로고 생성)

```text
이 클라이언트 혁신 서밋을 위한 심플하고 모던한 로고를 제작해 주세요.
```

### 설명 포인트

- 문장이 아닌 시간·세션명·설명이 포함된 일정표 형태로 결과 생성
- 텍스트 기획 + 시각적 아이디어(로고)까지 초안 제작
- 이미지 생성은 라이선스 없으면 횟수 제한 가능 → 데모 수준에서는 충분

## 데모 1-5. 서밋 기획 문서 작성

### 시나리오

지금까지 만든 내용을 1페이지 기획 브리핑 문서로 정리하고 Word로 생성

### 강사 진행 (순서대로 하나씩)

Prompt (기획 브리핑)

```text
이번 고객 혁신 서밋을 위한 1페이지 분량의 기획 브리핑을 작성하세요. 브리핑에는 목표, 참석자, 세션 주제, 주요 계획 마일스톤이 포함되어야 합니다.
```

후속 Prompt (성과·지표 섹션 추가)

```text
그리고, 행사에서 예상되는 성과와 성공 지표를 요약하는 섹션을 추가하세요.
```

후속 Prompt (Word 문서 생성)

```text
지금까지의 작성된 기획 브리핑을 워드 문서로 작성하세요.
```

### 설명 포인트

- 아이디어를 내부 공유·의사결정용 기획 문서로 정리
- 생성된 파일은 개인 OneDrive `Copilot에서 생성됨` 폴더에서 확인 (필요 시 로컬 다운로드)

## 데모 1-6. 파일 기반 분석 및 콘텐츠 생성

### 시나리오

앞에서 만든 Word 문서를 업로드해 요약하고, 후속 이메일 초안까지 생성

### 강사 진행

- OneDrive의 파일 또는 로컬에 다운로드한 파일을 첨부

Prompt (문서 요약)

```text
이 기획 브리핑의 핵심 사항을 5개의 간결한 불릿 포인트로 요약하세요.
```

후속 Prompt (후속 이메일 작성)

```text
요약된 핵심 내용과 다음 단계를 기획팀에 이메일로 전달하세요.
```

### 설명 포인트

- 문서 분량과 상관없이 핵심만 추려 요약
- 맥락 기반이라 영문 문서의 뉘앙스·의도 파악에 강점
- 요약 → 이메일 초안까지 이어져 작성 시간을 크게 단축

## 데모 1-7. Copilot Pages를 이용한 협업

### 시나리오

생성한 결과를 Copilot Pages로 옮겨 함께 편집·공유

### 강사 진행

- **응답 하단 버튼 안내**: `복사`(복붙 좋게/`md 포맷으로 작성해줘`), `피드백(👍/👎)`, `공유`(공유 링크 생성, 범위는 조직 정책·권한에 따름)
- 마지막 응답에서 **`페이지 내 편집`**을 선택해 Copilot 페이지로 복사
- 페이지에서: 제목·콘텐츠 추가, `/`로 콘텐츠 블록(글머리표 등) 삽입, 동료 `@mention`, 우측 상단 공유

Prompt (블록 수정 예시)

```text
행사에 초대할만한 연사 항목 추가해줘.
```

### 설명 포인트

- 편집 화면에서 **댓글**로 의견 교환
- **만들기**: 링크 전달, Word/PDF로 내보내기
- **페이지 링크** = 문서 자체 공유, **구성 요소 복사** = 내용을 Loop 컴포넌트로 복사해 Teams·Outlook·Word 등에서 함께 작업

## 데모 1-8. 에이전트 스토어 둘러보기

### 시나리오

미리 빌드된 Agent가 무엇인지, 어디서 찾는지 확인

### 강사 진행 (에이전트 스토어 구성)

- **내 에이전트**: 내가 추가했거나 자주 쓰는 에이전트 (에이전트 즐겨찾기)
- **Microsoft에서 빌드**: Microsoft가 직접 제공 (예: AI Learning Advisor, Prompt Coach)
- **추천**: 사용 환경·인기도 기준으로 추천되는 에이전트
- **더 많은 에이전트**: Microsoft + 파트너 제공 (예: Jira, Confluence, Adobe Acrobat, Workday) — 마켓플레이스 역할

### 설명 포인트

- 별도 구성·훈련 없이 바로 사용 가능한 전문 도우미
- 평소 말하듯 질문·지시하면 됨

## 데모 1-9. 미리 빌드된 Agent — PowerPoint

### 시나리오

기획 문서를 기반으로 PowerPoint 발표자료 초안 생성

### 강사 진행

- PowerPoint 에이전트 열기 → 앞에서 만든 Word 문서 추가

Prompt

```text
이 문서를 기반으로 7장 분량의 발표자료를 만들어 주세요. 표지, 목표, 산업 트렌드, 세션 소개, 일정표, 기대 효과, 마무리를 포함해 주세요.
```

- 추가 옵션 선택 화면이 나오면 원하는 스타일 선택 → 생성
- 생성 후 OneDrive 저장 또는 로컬 다운로드

### 설명 포인트

- 미리 빌드된 에이전트는 목적에 특화된 기능을 수행
- 생성에 시간·리소스가 많이 소요 → 수업에서는 미리 만들어 둔 결과로 진행 권장

## 데모 1-10. 미리 빌드된 Agent — Prompt Coach

### 시나리오

작성한 프롬프트를 분석해 더 좋은 결과를 얻도록 개선(프롬프트 엔지니어링) 코칭

### 강사 진행

- Prompt Coach 검색해서 열기 → 앞에서 쓴 프롬프트를 그대로 개선 요청

Prompt (기존 프롬프트 개선)

```text
반도체 트렌드를 주제로 한 1일 서밋의 일정표를 작성해 주세요.
서론, 마무리, 오전 9시, 11시, 오후 1시 30분, 3시 세션,
두 번의 짧은 휴식 시간, 그리고 점심 시간을 포함해
테이블 형식으로 작성해 주세요.

위 프롬프트를 분석하고,
더 구체적이고 효과적인 결과를 얻을 수 있도록 개선해 주세요.
개선 이유도 함께 설명해 주세요.
```

Prompt (새 세션 — 프롬프트 생성 도움)

```text
제주도 3박4일 가족 여행을 위한 상세 일정표 작성을 위한 프롬프트 생성 도와 주세요.
```

후속 Prompt (생성된 프롬프트 실행 후 수정)

```text
Day 1 저녁에 카페 대신에 와인바로 일정 업데이트 해주세요.
```

### 설명 포인트

- 프롬프트를 대신 실행하는 게 아니라, 부족한 점·추가하면 좋을 정보를 코칭
- 원본과 개선 프롬프트를 비교해 프롬프트 포맷을 정형화
- 미리 빌드된 에이전트는 새 채팅에서 불러 쓰거나 직접 열어서 사용 가능

## 데모 1 핵심 메시지

- Copilot Chat은 단순 검색 도구가 아니라 대화형 AI 도우미다.
- 웹의 최신 정보와 업로드한 문서를 결합해 리서치부터 기획 문서까지 이어서 만들 수 있다.
- 파일을 업로드하면 문서 기반 분석·요약·생성이 가능하다.
- Copilot Pages로 결과를 함께 편집·공유할 수 있다.
- Prebuilt Agent는 특정 업무 목적에 최적화된 Copilot 경험을 제공한다.
- 좋은 결과를 얻으려면 질문하는 방법(프롬프트)도 중요하다.
````
