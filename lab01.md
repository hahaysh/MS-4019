## 데모 1-1. Copilot Chat 기본 사용

### 참고 자료

#### Microsoft Learn

Explore Microsoft 365 Copilot Chat

https://learn.microsoft.com/en-us/training/modules/get-started-microsoft-365-copilot-business-chat/

### Learn Unit

- What is Microsoft 365 Copilot Chat?
- Use prompts effectively
- Continue conversations with follow-up prompts

### 시나리오

Copilot Chat이 일반 검색과 어떻게 다른지 확인

### 강사 진행

#### Prompt

```text

```

#### 후속 Prompt

```text
위 내용을 표 형식으로 정리해줘.
```

#### 후속 Prompt

```text
IT 관리자가 알아야 할 내용만 다시 정리해줘.
```

### 설명 포인트

- 대화 컨텍스트 유지
- 후속 질문 활용
- 검색이 아닌 대화형 AI

## 데모 1 핵심 메시지

- Copilot Chat은 단순 검색 도구가 아니라 대화형 AI 도우미이다.
- 파일을 업로드하면 문서 기반 분석과 요약이 가능하다.
- 웹 정보와 문서 정보를 함께 활용할 수 있다.
- Prebuilt Agent는 특정 업무 목적에 최적화된 Copilot 경험을 제공한다.
- 좋은 결과를 얻기 위해서는 질문하는 방법도 중요하다.
